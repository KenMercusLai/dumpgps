# Overview

dump gps info from photos

# Setup

## Requirements

* Python 3.6+

## Installation

Install dumpgps with pip:

```sh
$ pip install dumpgps
```

or directly from the source code:

```sh
$ git clone https://github.com/KenMercusLai/dumpgps.git
$ cd dumpgps
$ python setup.py install
```

# Usage

After installation, the package can imported:

```sh
$ python
>>> import dumpgps
>>> dumpgps.__version__
```
